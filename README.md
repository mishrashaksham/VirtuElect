# 🗳️ VirtuElect

**Next-Gen EVM Simulator & Manifesto AI Platform**  
*The future of secure, verifiable digital democracy.*

VirtuElect is a high-fidelity digital platform designed to simulate the Indian electoral process while empowering voters with AI-driven insights. From a fully tactile Electronic Voting Machine (EVM) simulator to an intelligent RAG-based AI assistant, VirtuElect bridges the gap between technology and transparent democracy.

---

## ✨ Key Features

*   **🕹️ High-Fidelity EVM Simulator:** A realistic, interactive ballot unit featuring tactile blue voting buttons, authentic candidate rows, and immediate physical VVPAT (Voter Verifiable Paper Audit Trail) generation.
*   **🧠 Gyani (Manifesto RAG AI):** Powered by Google's Gemini Models (`gemini-2.5-flash` & `gemini-embedding-2`), this in-memory Vector RAG system allows users to ask direct questions about various political party manifestos. 
*   **📊 Know Your Candidate:** Explore AI-summarized affidavits, financial backgrounds, and criminal records of candidates before casting a simulated vote.
*   **🗺️ Interactive Voter Journey:** A step-by-step 3D animated timeline guiding users through Registration, KYC, EVM Voting, and VVPAT Verification.
*   **🌌 Premium Dark UI:** Built with Tailwind CSS v4, featuring a sleek dark-mode exclusive design, custom Neumorphic shadows, and a dynamic, performance-optimized glowing cursor aura.

---

## 🛠️ Tech Stack

**Frontend:**
*   React.js
*   Tailwind CSS v4 (Custom UI, Neumorphism, Glassmorphism)
*   Vite

**Backend & AI:**
*   Node.js & Express.js
*   Google Generative AI SDK (Gemini API)
*   Custom In-Memory Vector Store for RAG (Retrieval-Augmented Generation)
